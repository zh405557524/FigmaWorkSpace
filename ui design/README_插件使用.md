## 🚀 3步开始使用

### 1️⃣ 打开插件
在 Figma 桌面应用中：
- 点击 `Plugins` → `Development` → `Import plugin from manifest`
- 选择 `ui design/code.js`

### 2️⃣ 选择功能
点击你需要的按钮：
- **首页** - 完整的应用首页
- **学习进度** - 数据可视化页面
- **设置页面** - 用户设置和账户管理 ⭐ 最新
- **练习模式** - 练习模式列表
- **钢琴键盘** - 钢琴键盘组件
- **设计示例** - 颜色和字体系统

### 3️⃣ 完成！
等待几秒，设计稿会自动生成在画布上

## 💡 使用技巧

### ✅ 推荐做法
1. **先生成，再调整** - 先用插件生成基础设计稿，然后在 Figma 中微调
2. **批量生成** - 可以多次运行插件，生成多个版本对比
3. **自定义数据** - 编辑 `code.js` 中的数据变量，生成真实数据的设计稿

### ⚠️ 注意事项
1. **必须使用桌面应用** - 浏览器版 Figma 不支持开发者插件
2. **需要 Inter 字体** - 确保系统已安装（通常都有）
3. **每次生成新的** - 插件会创建新的框架，不会覆盖现有设计

## 🐛 常见问题

### Q: 插件按钮点击后没反应？
A: 检查 Figma 控制台是否有错误（`Plugins` → `Development` → `Open Console`）

### Q: 字体显示不正确？
A: 确保系统已安装 Inter 字体，或者修改代码使用其他字体

### Q: 想修改设计稿的数据？
A: 编辑 `code.js` 文件，搜索对应的数据变量（如 `statsData`、`dailyScores`）

### Q: 可以导出给开发使用吗？
A: 可以！生成的设计稿完全遵循设计规范，可以直接导出给开发团队
