## 🚀 使用方法

### 方式一：从统一插件运行

1. 打开 Figma 桌面应用
2. 进入你的设计文件
3. 打开插件：`Plugins` → `Development` → `Import plugin from manifest`
4. 选择 `ui design/code.js`
5. 在插件界面点击 **"⚙️ 创建设置页面"** 按钮
6. 等待几秒，设计稿自动生成

### 方式二：运行独立插件

1. 打开 Figma 桌面应用
2. 进入你的设计文件
3. `Plugins` → `Development` → `Import plugin from manifest`
4. 选择 `ui design/create-settings-page.js`
5. 点击 "创建设计稿" 按钮

## 🎯 设计亮点

1. **个性化头像**：渐变圆形头像，突出用户身份
2. **分类清晰**：练习设置和账户帮助分区明确
3. **视觉层次**：使用不同背景色区分功能类别
4. **一致性**：与首页、进度页面保持统一的设计语言
5. **可操作性**：每个项目都有右箭头，暗示可点击

## 🎨 设计规范

本页面完全遵循 `ui设计稿生成规范.md` 中的设计标准：

### 颜色使用
- ✅ SOLID 类型颜色只包含 `{ r, g, b }`
- ✅ 渐变颜色包含 `{ r, g, b, a: 1 }`
- ✅ 渐变包含 `gradientTransform`

### 文字创建
- ✅ 先加载字体 → 先设置 fontName → 设置 fontSize → 最后设置 characters
- ✅ 使用 Inter 字体，支持 Regular 和 Medium 字重

### 错误处理
- ✅ 使用 try-catch 包裹所有代码
- ✅ 提供友好的错误提示
- ✅ 记录详细错误信息到控制台

### 代码组织
- ✅ 所有代码内联，不使用 require/import
- ✅ 颜色、字体等设计令牌直接定义在文件中

## 💡 开发建议

### 实现交互
在实际开发中，每个设置项应该：
1. **音符与练习** - 跳转到音符范围、调号选择页面
2. **音效设置** - 切换音效开关，支持开/关状态
3. **练习提醒** - 设置提醒时间，支持推送通知
4. **账户信息** - 跳转到个人资料编辑页面
5. **帮助中心** - 显示常见问题列表或跳转到帮助页面
6. **关于应用** - 显示应用详细信息、许可协议等

### 状态管理
考虑添加：
- 用户等级徽章
- 成就展示
- 练习统计快速预览
- 设置同步状态指示

### 扩展功能
可以添加：
- 主题切换（亮色/暗色）
- 语言设置
- 字体大小调整
- 数据导出/备份
